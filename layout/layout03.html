        方式一： 左右浮动, 中间区域自适应

        浮动布局
        先来设置左、右栏目浮动到左右两边, 因为是浮动会脱离文档流, 则内容容器会撑满全屏
        内容容器只需要设置margin的left和right留出空间即可 
        
        优点: 简单 - 结果固定, 内容容器必须放在后面

        缺点: 最后加载, 内容多时可能影响体验

        .m-left, .m-right{
            width: 200px; height: 100%;
            background: #cbfafc;
        }
        .m-left{
            float: left; 
        }
        .m-right{
            float: right;
        }
        .m-cont{
            background: #fac9c5;
            min-height: 100%;
            margin: 0 200px;
        }
